# Emulação de Rede P2P/BitTorrent com Kathara

Este repositório contém os arquivos de configuração e scripts utilizados para emular uma arquitetura simplificada de compartilhamento de arquivos P2P (BitTorrent) utilizando o framework Kathara (Docker).

## Como reproduzir o experimento

### Pré-requisitos
* Sistema Operacional com WSL2 (Ubuntu) ou Linux.
* Docker instalado e rodando.
* Framework Kathara instalado.

### Passo a Passo para Execução
